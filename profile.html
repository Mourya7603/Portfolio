My Journey

How I built this portfolio and what I learned

I created this portfolio as part of my learning journey in becoming a MERN Stack Developer. Starting with basic HTML and CSS, I explored semantic UI frameworks like PicoCSS to create clean and accessible layouts. The project helped me understand responsive design, JavaScript interactivity (like the dark/light mode toggle), and how to organize and showcase my work.

One of the most rewarding parts was learning how to break complex projects into smaller micro tools, such as the Fitness Toolbox or Marketing Utilities. Each one pushed me further in my understanding of frontend-backend communication, component-based design, and Git version control.
